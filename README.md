Waterbill — это специализированный пакет на языке Go, предназначенный для автоматизации расчетов, связанных с потреблением воды и формированием квитанций. Он предоставляет простой и надежный набор функций для решения типовых задач жилищно-коммунального хозяйства (ЖКХ) и управления водными ресурсами.

Пакет разработан с акцентом на:

Надежность — тщательная проверка входных данных и обработка ошибок.

Прозрачность — понятные функции для каждого этапа расчета.

Гибкость — возможность применения штрафов и форматирования отчетов.
